pierluc.io

Le site Web sur lequel vous vous trouvez est une simple page statique et surtout une vitrine technologique offrant une architecture dernier cri.

Cette infrastructure comporte plusieurs avantages : déploiement et mises à jour en un clic, temps de chargement rapide, aucun point de défaillance unique et un faible coût d'opération.

Technologies
  • Jekyll - générateur de site statique
  • GitHub - contrôle de source
  • Docker Cloud - déploiement de conteneurs
  • Azure - équilibrage de charge, machines virtuelles et stockage
  • haproxy - terminaison SSL/TLS et proxy
  • nginx - serveur Web
  • Cloudflare - réseau de diffusion de contenu (CDN)
  • Apex Ping - suivi de disponibilité et performance
